The Mod Bot Beta 1

From a Hackathon Winning Project to SafeComms Incorporated's first product launch, the Mod Bot Beta 1 brings Moderation to a new level. A Bot initially designed to run on Pronto's API, The Mod Bot uses an extremely accurate websocket that parses its raw data and automatically searches for it's channel code without the assistance of the user. Using this web socket, it scans every message sent on a specific chat, which then logs in a different chat where human moderators could retrieve messages sent that may violate policies. Spam detection and Profanity has been added which will give 3 warnings. After all warnings have been issued, the user will be removed from said chat and be dmed and auto generated messages giving the reason why the user was removed. The Mod Bot Beta 1 not only one the 2025 FTC Robotics/APC Hackathon, but has impressed many students and faculty members in the process, leading the age of mods to go from human disciplinarians to bots.
