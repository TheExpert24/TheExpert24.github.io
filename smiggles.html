Smiggles

Smiggles is a lightweight operating system coded primarily from C and Assembly. It boots on GRUB and MemoryBoot2, meaning you could run it on real hardware. It has a file system, networking, a package installer, basic interpreter, command line, and many other cool features that make up an operating system. My friend Jules Miller and I having been working on this project for 3 months. Sitting at over 15,000 lines of code, this behemoth is a massive headache to work on, but we adamant to topple the regime built by Apple, Microsoft, and Linux, with Smiggles.
